# Noções básicas do [!UICONTROL Journey Orchestration]

## Introdução ao [!UICONTROL Journey Orchestration]

[!UICONTROL Journey Orchestration] O permite criar casos de uso de orquestração em tempo real usando dados contextuais armazenados em eventos ou fontes de dados.

O [!UICONTROL Journey Orchestration] é um serviço de aplicativos integrado à Adobe Experience Platform. Ele fornece um ambiente inteligente e aberto para ativar todos os dados relevantes por meio de envolvimento escalável e baseado em eventos em qualquer canal que sua empresa exija, seja marketing, operações ou serviços. [!UICONTROL Journey Orchestration] O pode usar qualquer dado da Adobe Experience Platform e de qualquer sistema de delivery externo para criar e fornecer experiências atraentes.

## Como configurar uma jornada

As principais etapas de preparação para as jornadas de construção são:

1. [Configurar eventos de transmissão](/help/configuring-journey-orchestration/configure-streaming-events.md) - essa configuração é obrigatória, pois o [!UICONTROL Journey Orchestration] é projetado para ouvir eventos.
1. [Configurar fontes de dados](/help/configuring-journey-orchestration/configure-data-sources.md)  - essa configuração não é necessária se suas jornadas usarem apenas dados locais provenientes de uma carga útil do evento.
1. [Configurar ações personalizadas](/help/configuring-journey-orchestration/configure-actions.md): é necessário caso você queira usar um serviço de qualquer provedor de terceiros que possa ser chamado por meio de uma [!DNL REST API] com uma carga de formato JSON

>[!NOTE]
>
>Essas etapas de configuração exigem conhecimento técnico. Você deve estar familiarizado com o [Experience Data Model (XDM)](https://experienceleague.adobe.com/docs/platform-learn/tutorials/schemas/schemas-and-experience-data-model.html?lang=en) e [como compor schemas de evento de experiência XDM](https://experienceleague.adobe.com/docs/platform-learn/tutorials/schemas/create-schemas.html?lang=en).
